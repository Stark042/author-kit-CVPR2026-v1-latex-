\section{Methodology and Experiments}
\label{sec:methodology_experiments}

\subsection{Methodology}

We investigate the effect of training-time exposure to out-of-distribution (OoD) data on the behavior of object detectors. Our methodology consists of three main stages: (i) training a baseline object detector on in-distribution (ID) data, (ii) fine-tuning the model using OoD samples treated as background, and (iii) analyzing the resulting models using both quantitative metrics and explainability techniques. The overall goal is to understand how fine-tuning alters model behavior and contributes to reducing hallucinated detections on OoD inputs.

For in-distribution training, we use the PASCAL VOC 2012 dataset, which contains annotated object categories commonly used for benchmarking object detection models. The dataset is converted into YOLO format by transforming XML annotations into normalized bounding box representations. A train–validation split is created using standard data partitioning.

For OoD data, we utilize the Caltech 256 dataset, which contains a diverse set of object categories that do not overlap with the ID classes. A subset of 8,140 images is used for fine-tuning. Importantly, these OoD samples are treated as background during training, meaning no foreground object labels are assigned. This ensures that the model learns to suppress objectness scores for semantically similar but out-of-distribution inputs.

We employ the YOLOv10s architecture implemented using the Ultralytics YOLO framework. The baseline model is trained on the ID dataset for 50 epochs with an input resolution of $640\times640$, batch size of 64, and default optimization settings. Training is performed using multiple GPUs to accelerate convergence.

The fine-tuned model is initialized from the trained baseline and further trained for 20 epochs using a combined dataset consisting of ID data and OoD samples. During fine-tuning, the input resolution is set to $512\times512$ with a batch size of 32. The standard YOLO loss function is used without modification.

The key idea of our approach is to expose the model to OoD samples during training and explicitly treat them as background. Unlike traditional OoD detection methods that operate at inference time, this strategy modifies the model's internal representation during training. By assigning OoD samples to the background class, the model is encouraged to produce low objectness scores for regions that do not correspond to valid ID objects.

This process effectively reshapes the decision boundary of the detector. In standard training, the model learns to distinguish foreground objects from background based solely on ID data, leaving OoD regions underconstrained. Fine-tuning introduces additional constraints by penalizing object-like responses on OoD inputs, resulting in a more conservative objectness function. This reduces the likelihood of hallucinated detections when the model encounters unseen data.

To evaluate model behavior under distribution shift, we perform inference on OoD test images using both baseline and fine-tuned models. Predictions are generated with a confidence threshold of 0.25. We measure hallucinated detections by analyzing the number and quality of predicted bounding boxes on OoD inputs.

To assess spatial alignment and relevance, we compute Intersection-over-Union (IoU) between predicted regions and salient regions identified by explainability methods. IoU is evaluated at multiple thresholds (10\%, 20\%, and 30\%) to capture varying levels of localization accuracy.

To understand the behavioral differences between the baseline and fine-tuned models, we employ multiple explainability techniques.

\paragraph{Grad-CAM.}
We use Gradient-weighted Class Activation Mapping (Grad-CAM) to visualize the regions of the input image that contribute most to the model's predictions. Grad-CAM is applied to the last convolutional layer of the network using a customized wrapper compatible with the YOLO architecture. This allows us to analyze whether the model focuses on semantically meaningful regions or irrelevant background features.

\paragraph{Occlusion Analysis.}
We perform occlusion-based analysis by systematically masking parts of the input image and observing the resulting change in prediction confidence. This helps identify which regions are critical for the model's decision-making process.

\paragraph{Insertion and Deletion Metrics.}
We quantify explanation quality using insertion and deletion curves. The insertion metric measures how quickly the model's confidence increases as important regions are progressively revealed, while the deletion metric measures how rapidly confidence decreases as important regions are removed. These metrics provide a quantitative assessment of how well the model's attention aligns with meaningful features.

\paragraph{Pointing Game.}
We further evaluate localization quality using the pointing game metric, which checks whether the most salient point in the Grad-CAM heatmap falls within the predicted object region.

\paragraph{Frequency Bias Analysis.}
To analyze potential biases, we examine the distribution of predicted classes and identify whether the model exhibits a tendency to favor frequently occurring categories. This helps assess whether fine-tuning reduces reliance on spurious correlations present in the training data.

While the fine-tuning strategy improves robustness by reshaping the detector's decision boundary, it does not explicitly address the role of low-level signal characteristics at inference time. To further mitigate hallucinated detections, we introduce a lightweight post-hoc calibration mechanism applied directly to model predictions.

\paragraph{Motivation}
Empirical observations from our experiments indicate that:
\begin{itemize}
    \item High-frequency regions (e.g., textured backgrounds) frequently trigger false detections,
    \item Small bounding boxes exhibit unstable and overconfident predictions,
    \item These effects are amplified in OoD settings.
\end{itemize}

This behavior aligns with the known spectral bias of deep neural networks, where models tend to rely on high-frequency patterns.

\paragraph{Frequency Estimation}
Given an image patch $x \in \mathbb{R}^{H\times W}$ associated with a detection, we estimate high-frequency content using the discrete Laplacian response:
\begin{equation}
    \Delta x(p) = \sum_{q \in \mathcal{N}(p)} x(q) - 4x(p)
\end{equation}
where $\mathcal{N}(p)$ denotes the 4-neighborhood of pixel $p$. The raw frequency energy is then defined as
\begin{equation}
    e_f(x) = \frac{1}{|\Omega|} \sum_{p\in\Omega} |\Delta x(p)|^2,
\end{equation}
with $\Omega$ the patch domain. This quantity is proportional to the spectral second moment and emphasizes rapid spatial oscillations (edges/textures), which are frequently associated with OoD hallucination triggers.

To make the measure comparable across images, we apply robust dataset-level normalization:
\begin{equation}
    z_f(x) = \frac{e_f(x) - \operatorname{med}(e_f)}{\operatorname{MAD}(e_f) + \varepsilon},
\end{equation}
where $\operatorname{med}(\cdot)$ and $\operatorname{MAD}(\cdot)$ are the median and median absolute deviation over the calibration set. Finally, we map to a bounded penalty term:
\begin{equation}
    \hat{f}(x) = \min\!\left(\max\!\left(\sigma\!\left(z_f(x)\right),\,0\right),\,1\right), \quad
    \sigma(t)=\frac{1}{1+e^{-t}}.
\end{equation}
This bounded form prevents extreme suppression from outliers while preserving rank ordering of high-frequency detections.

\paragraph{Objectness Approximation}
For one-stage detectors, the final confidence is approximately factorized as
\begin{equation}
    s \approx p_{\text{obj}}\, p_{\text{cls}|\text{obj}},
\end{equation}
where $p_{\text{obj}}$ is objectness and $p_{\text{cls}|\text{obj}}$ is class posterior. Since explicit $p_{\text{obj}}$ is not always exposed in exported predictions, we construct a monotonic proxy from confidence:
\begin{equation}
    o(s) = s^{\eta}, \qquad 0<\eta<1,
\end{equation}
and use $\eta=\tfrac{1}{2}$ in our experiments. The sublinear transform compresses very high confidences while preserving ordering, making suppression smoother for borderline detections. We further use a bounded form
\begin{equation}
    \hat{o}(s)=\min\!\left(\max\!\left(\frac{o(s)-\mu_o}{\sigma_o+\varepsilon},\,0\right),\,1\right),
\end{equation}
where $\mu_o,\sigma_o$ are mean and standard deviation of $o(s)$ estimated on validation predictions. This yields a stable, scale-aware objectness penalty for post-hoc calibration.

\paragraph{Calibration Function}
For each detection with confidence $s$, we compute the calibrated score:
\begin{equation}
    s' = s \cdot \exp(-\alpha \hat{f}) \cdot \exp(-\beta o) \cdot \frac{1}{\gamma(a)}
\end{equation}
where:
\begin{itemize}
    \item $\alpha$ controls frequency suppression,
    \item $\beta$ controls objectness suppression,
    \item $a$ is bounding box area.
\end{itemize}
The size penalty is defined as:
\begin{equation}
    \gamma(a) =
    \begin{cases}
        \lambda, & a < \tau \\
        1, & \text{otherwise}
    \end{cases}
\end{equation}

\paragraph{Stability Constraint}
To avoid artificial confidence boosting:
\begin{equation}
    s' = \min(s', s)
\end{equation}
This ensures calibration only suppresses overconfident predictions.

\paragraph{Variants}
We evaluate two variants:
\begin{itemize}
    \item \textbf{Frequency-only calibration}: $s' = s \cdot \exp(-\alpha \hat{f}) / \gamma(a)$,
    \item \textbf{Frequency + Objectness calibration}: full formulation above.
\end{itemize}

\paragraph{Implementation Details}
The method is applied post-hoc on detector outputs, requires no retraining or architecture modification, is computationally lightweight (Laplacian + scalar ops), and is compatible with real-time object detectors (YOLO variants).

\paragraph{Key Insight}
This formulation explicitly integrates frequency-domain cues into the detection pipeline, complementing training-time OoD exposure and providing an additional mechanism for suppressing hallucinated detections.

\subsection{Experiment 2}

\paragraph{Experimental Setup.}
We evaluate the proposed frequency-aware objectness calibration framework on both in-distribution (ID) and out-of-distribution (OoD) data. ID performance is measured on the Pascal VOC 2012 validation split, while OoD behavior is evaluated on a proximal OoD benchmark constructed from Caltech-256 categories that are visually similar yet semantically mismatched with VOC classes.

Inference is performed with a fine-tuned YOLO detector, and calibration is applied strictly as a post-processing step (no retraining and no architecture modification). Unless stated otherwise, predictions are filtered with confidence threshold $\tau=0.25$.

\paragraph{Evaluation Metrics.}
We report three OoD metrics:
\begin{itemize}
    \item \textbf{Hallucination Rate (HR):} fraction of images with at least one detection above threshold.
    \item \textbf{False Positive Count (FP):} total number of detections on the OoD set.
    \item \textbf{Reduction (\%):} relative FP drop after calibration,
\end{itemize}
\begin{equation}
    \mathrm{Reduction}(\%) = \frac{\mathrm{FP}_{\mathrm{raw}}-\mathrm{FP}_{\mathrm{cal}}}{\mathrm{FP}_{\mathrm{raw}}}\times 100.
\end{equation}

\paragraph{Quantitative Results.}
Relative to the baseline ($\mathrm{FP}=444$, $\mathrm{HR}=0.2096$), frequency-only calibration reduces false positives to 389 and HR to 0.1899, corresponding to a $12.38\%$ reduction. Adding objectness-aware suppression yields a larger gain: FP decreases to 324 and HR to 0.1641, achieving $27.03\%$ total reduction. These results indicate that frequency cues remove a substantial portion of spurious detections, while objectness modulation further suppresses overconfident false alarms.

\paragraph{Qualitative Results.}
Visual inspection shows that baseline predictions frequently include spurious boxes in cluttered and high-texture regions, whereas the calibrated model yields cleaner detections and more stable localization. In particular, frequency + objectness calibration reduces scattered small-box artifacts and improves alignment with semantically meaningful structures.

\paragraph{Confidence Calibration Analysis.}
We observe a consistent left-shift in confidence after calibration:
\begin{equation}
    \Delta\mu = \mathbb{E}[\mathcal{S}_{\mathrm{raw}}]-\mathbb{E}[\mathcal{S}_{\mathrm{cal}}] > 0,
\end{equation}
indicating suppression of overconfident OoD detections. This behavior is stable across IoU thresholds (10\%, 20\%, 30\%), suggesting that the score correction is not due to random score collapse but targeted attenuation of unreliable predictions.

\paragraph{Frequency--Objectness Relationship.}
To verify the calibration hypothesis, we analyze the coupling between frequency energy and confidence-derived objectness proxy. Empirically, detections from high-frequency regions tend to have disproportionately high objectness estimates, supporting the presence of frequency-induced bias in one-stage detectors. This validates the need for a joint penalty that combines spectral and objectness information.

To understand model behavior, we employ Grad-CAM, occlusion analysis, insertion/deletion metrics, pointing game, and frequency bias analysis. Grad-CAM highlights the most salient regions, occlusion reveals prediction sensitivity, insertion/deletion quantify explanation quality, pointing game measures localization, and frequency bias analysis checks whether the model relies on commonly occurring classes.

\paragraph{Discussion.}
Overall, the results demonstrate that: (i) frequency-aware suppression reduces texture-induced false detections, (ii) objectness-aware modulation provides additional stability and further lowers hallucination rate, and (iii) the combined method achieves up to $27.03\%$ FP reduction with minimal computational overhead. The approach is training-free at inference time, easy to integrate, and compatible with real-time YOLO-style detectors.
